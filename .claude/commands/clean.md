Lint, format, and typecheck the project. Fix any issues found.

1. Run `npm run format` to auto-format with biome
2. Run `npm run lint` to lint with biome and auto-fix what's possible
3. Run `npm run typecheck` to check for TypeScript errors
4. If there are remaining lint or type errors that couldn't be auto-fixed, read the affected files and fix them manually
5. Re-run the failing step to confirm the fix
6. Summarize what was fixed
